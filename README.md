🍕 Pizza Sales Data Analysis (SQL)
📌 Project Overview
This project focuses on analyzing a comprehensive pizza sales dataset to uncover key business insights. Using Advanced SQL queries, I explored trends in revenue, customer behavior, and product performance to provide data-driven recommendations.
🛠️ Tech Stack
Database: MySQL
Analysis Tool: MySQL Workbench
Concepts Used: Joins, CTEs, Window Functions (RANK() OVER, SUM() OVER), Subqueries, and Data Aggregation.
📊 Key Business Questions Solved
Revenue Analysis: Calculated total revenue and identified the highest-priced pizza.
Customer Insights: Found the most common pizza sizes and total order volume.
Performance Metrics: Identified the Top 5 best-selling pizzas and category-wise sales distribution.
Advanced Analytics: Analyzed cumulative revenue over time and identified top 3 pizzas in each category.
